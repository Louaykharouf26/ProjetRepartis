**** Développement d’un système de partage de fichiers **** 
l'execution de l'application se fait en executant la classe main en java 

**Build de l'application ** 
javac main.java 
=> permet le build des classes 
**Execution de l'application ** 
java main.java 
l'utilisateur choisi soit 1 soit 2 
s'il choisi 1 la connexion est sous type serveur 
s'il choisi 2 la connexion est sous type client 
le client entre le chemin vers le dossier ensuite le port de connexion et le peerID et le fichier qu'il souhaite telecharger
